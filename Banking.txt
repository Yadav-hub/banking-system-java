 Mini Banking System — Challenge
Build these classes:

1. BankAccount class

Fields — accountNumber, accountHolderName, balance
Methods —

deposit(double amount) — add money, throw exception if amount ≤ 0
withdraw(double amount) — deduct money, throw exception if insufficient balance or amount ≤ 0
getBalance() — return current balance
printStatement() — print account details nicely




2. Custom Exceptions

InsufficientFundsException — when withdrawal exceeds balance
InvalidAmountException — when amount is 0 or negative


3. Bank class

Use a HashMap<String, BankAccount> to store accounts — key is account number
Methods —

createAccount(String accNo, String name, double initialDeposit)
getAccount(String accNo) — return account, throw exception if not found
transferFunds(String fromAcc, String toAcc, double amount) — withdraw from one, deposit to another




4. Main class

Create 3 accounts
Do deposits, withdrawals
Transfer between accounts
Deliberately trigger exceptions and handle them with proper messages
